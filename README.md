# Movie Recommendation Agent

Agent IA de recommandation de films basé sur :
- LangChain (agents + tools)
- FAISS (similarité sémantique)
- Modèle PyTorch (classification de genre)
- API Flask (backend)
- Streamlit (frontend)
- Docker & Docker Compose

## Installation

- Cloner le projet
- Créer une API Key sur open router et la mettre dans le fichier agent
- docker-compose build


## Lancement
docker-compose up
